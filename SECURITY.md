# 安全策略

## 报告安全漏洞

我们非常重视安全问题。如果您发现任何安全漏洞，请不要在公开的 Issue 中讨论，而是直接发送邮件到 momail@vip.qq.com。

## 响应时间

我们会在24小时内回应您的报告，并会：

1. 确认漏洞的存在
2. 评估漏洞的严重程度
3. 制定修复计划
4. 通知您修复进展

## 漏洞处理流程

1. 收到报告后，我们会确认是否属于安全漏洞
2. 如果确认是安全漏洞：
   - 立即着手修复
   - 准备安全更新
   - 发布修复版本
   - 在修复后公开披露（如果需要）

## 安全最佳实践

1. 定期更新依赖
2. 使用安全的通信协议
3. 遵循安全编码规范
4. 进行安全测试
5. 保护敏感信息

## 奖励计划

对于帮助我们发现重要安全漏洞的贡献者，我们会：

1. 在我们的贡献者名单中特别致谢
2. 提供详细的技术反馈
3. 在修复后的发布说明中提及（如果您同意）
